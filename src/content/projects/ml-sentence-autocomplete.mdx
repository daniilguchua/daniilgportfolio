## Overview

A sentence autocomplete system powered by custom n-gram language models, built from the ground up without relying on pre-trained models. The system tokenizes input, analyzes context, and generates probabilistically ranked completions in real-time.

## Architecture

- **Tokenization pipeline**: Custom preprocessing with NLTK and spaCy for robust text normalization
- **N-gram models**: Bigram and trigram models with Laplace smoothing for handling unseen sequences
- **Flask API**: RESTful backend serving predictions with sub-100ms latency
- **Interactive frontend**: Real-time suggestion dropdown as users type

## Results

The system demonstrates strong completion accuracy on conversational English text, with the trigram model significantly outperforming the bigram baseline.
